# Ben 10 Field Notebook UI

A stylized, page-turning notebook UI built with React and Vite. It showcases the full Ben 10 roster with locally cached character portraits and a realistic notebook spread.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
